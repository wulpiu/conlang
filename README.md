# Conlang Studio

Десктопное приложение (PyQt6) для конструирования вымышленных языков (conlang) —
от фонологии и морфологии до синтаксиса и письменности.

## Возможности

- **Фонология** — инвентарь гласных/согласных, фонотактика, просодия
- **Морфология** — единая система парадигм по частям речи, грамматические категории
- **Лексикон** — словарь с леммами, частями речи, словообразованием
- **Синтаксис** — порядок слов, эргативность/аккузативность, согласование
- **Прагматика и письменность** — дейксис, регистры, система письма
- **Leipzig glossing** — разбор и генерация примеров с интерлинеарной глоссой
- **DNA-экспорт** — сохранение всего языка одной строкой (для обмена/бэкапа) и импорт обратно
- Встроенные пресеты: русский, английский, баскский

## Установка

```bash
pip install -r requirements.txt
```

Опционально, для анализа текста на естественных языках:
- `pymorphy3` — морфология русского
- `spacy` + модель `en_core_web_sm` — английский
- `stanza` — баскский

## Запуск

```bash
python main.py
```

## Разработка

Проект использует:
- `pytest` — тесты (round-trip сериализации модели данных и DNA-строк)
- `ruff` — линтер и форматтер
- `mypy` — проверка типов

```bash
pytest tests/ -v
ruff check .
mypy . --ignore-missing-imports --check-untyped-defs
```

## Структура проекта

```
core/       — модель данных, сериализация, общие виджеты
sections/   — разделы интерфейса (фонология, лексикон, синтаксис и т.д.)
tests/      — тесты
```
